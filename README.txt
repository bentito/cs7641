Code usage notes for Randomized Optimization Assignment

To get the code (now public w00t!):
git clone https://github.com/bentito/cs7641.git

The code is in the file https://github.com/bentito/cs7641/Assignment3CS7641.py


To recreate the Python environment:
cd cs7641
python -m pip install -r requirements.txt

run:
Assignment3CS7641.py <dataset_name>

where: dataset_name = "faces" or "forest", or leave off data set name to run both
